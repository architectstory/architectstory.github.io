## Why Micro Frontend?
  - 배경
    - Web application 의 발전
      - Web Programming, Social Web, Font-end 와 Back-end 분리
    - Microservice
    - Web 표준의 분상
      - Web component isolation
      - Frame communication
      - Web Worker & Web Proxy
    - 출시 기간 단축
      - 조직 적응 시간 단축, 여러 개의 팀, 피처 격리, A/B 테스팅
